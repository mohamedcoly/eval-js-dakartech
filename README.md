Description du projet

Ce projet est une application de liste de tâches (To-Do List) développée en HTML, CSS et JavaScript pur. L'application permet aux utilisateurs de gérer leurs tâches quotidiennes en les ajoutant, les marquant comme terminées, les supprimant et en filtrant l'affichage selon leur statut. Les données sont sauvegardées automatiquement dans le navigateur grâce au localStorage.

Fonctionnalités du to-do list

Formulaire d'ajout de tâche : champ texte et bouton "Ajouter"
Validation du champ : impossible d'ajouter une tâche vide avec message d'erreur
Liste des tâches: affichage sous forme de cartes
Bouton "Terminé" : marque une tâche comme terminée (texte barré, style grisé)
Bouton "Annuler" : remet une tâche en cours
Bouton "Supprimer : retire définitivement une tâche
Compteur dynamique: affiche le nombre de tâches en cours
Filtres : Toutes / En cours / Terminées (avec mise en évidence du filtre actif)
Sauvegarde localStorage : les tâches persistent après fermeture/rafraîchissement

 
 Technologies utilisées
HTML5 structure sémantique de la page
CSS3styles, mise en page responsive, animations
JavaScript logique métier, manipulation du DOM, événements, localStorage
